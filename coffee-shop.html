Project Overview

My primary focus while creating this landing page was to combine smooth animations, responsive design, and advanced scrolling effects.

Technologies used:

  • - Next.js and React: Essential for structuring the application and managing dynamic components.
  • - Tailwind CSS: A CSS framework that enabled responsive and modern design, boosting styling efficiency.
  • - Locomotive Scroll: First-time usage allowed me to explore and implement smooth scrolling effects, adding a unique, immersive touch to the landing page.
  • - Swiper: Provided a reliable framework for implementing responsive sliders.
  • - Git and Vercel: Used for version control and continuous deployment, with direct integration for streamlined development and production deployment.

This project was an excellent opportunity to familiarize myself with Locomotive Scroll and Framer Motion, learning how to integrate them into a project. Revisiting familiar tools like Swiper, React, and Tailwind reinforced my knowledge while enhancing my ability to combine them with new frameworks.

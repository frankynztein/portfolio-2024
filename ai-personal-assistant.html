AI Personal Assistant

An AI agent that can help you with common personal management tasks.

Project Overview

This project is a virtual assistant that I developed using N8N as an automation platform. Its main goal is to demonstrate the orchestration of workflows and the integration of multiple services and APIs to automate common personal tasks directly from Telegram.

Key Capabilities:

  • - Schedule management: add events to Google Calendar via chat commands.
  • - Centralized communication: Send emails through the Gmail API.
  • - Information access: answer general queries using OpenAI APIs (Chat Completion) and perform real-time searches via Serp API, Wikipedia, and Hacker News.
  • - Multimodal interaction: supports both text and voice, processing audio with OpenAI Whisper (Speech-to-Text) and generating spoken responses with OpenAI Text-to-Speech.

Technology and approach:

The architecture is centered on N8N to manage the business logic and calls to various external APIs. The Telegram Bot API serves as the main interface. I implemented modular flows in N8N to handle different user intents, routing requests to the corresponding integrations (Google APIs, OpenAI APIs, search APIs). I also used Google Sheets as a simple key data store.

For me, building this assistant meant overcoming several key challenges in workflow orchestration. One of the primary hurdles was getting the AI to accurately interpret diverse user intent and the correct order of actions; this required investing significant time in engineering and iterating on the model's "system prompt". Configuring secure connections to Google services (Gmail, Calendar, Sheets) from my VPS also added complexity, necessitating detailed credential management via the Google Cloud Console. Furthermore, handling both text and voice inputs from Telegram involved carefully designing the conditional logic with N8N's Switch node for correct routing. Overcoming these points was fundamental to achieving a functional and robust assistant.

Note on live demo: due to the usage-based costs associated with the OpenAI API credits, I cannot maintain a live, publicly accessible version of this assistant running continuously. However, I can provide a video demonstration or screenshots showcasing its functionalities upon request.

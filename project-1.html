PhotoSplash

This is an interactive web application designed to explore and discover high-quality images from the Unsplash platform.

Live Link

Project Overview

This project was instrumental in helping me learn the fundamentals of Next.js and develop a strong understanding of workflows with RESTful APIs.

Technologies used:

  • - Next.js: A React framework that facilitates building dynamic and SEO-friendly web applications.
  • - Unsplash API: Provides access to an extensive image library, allowing users to search by specific themes.
  • - Tailwind CSS: A CSS framework that enabled responsive and modern design, boosting styling efficiency.
  • - HTML, CSS and JavaScript: For the structure, functionality, and overall styling of the application.
  • - Git and Vercel: Used for version control and continuous deployment, with direct integration for streamlined development and production deployment.

Through PhotoSplash, I gained foundational knowledge in Next.js, image optimization, and API consumption, along with improving my workflow using Git and deploying on Vercel.
